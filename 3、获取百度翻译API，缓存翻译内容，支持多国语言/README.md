## 使用DEMO如下：


```
require 'vendor/Api.php';

// 使用DEMO
$obj = new Fanyi();
$res = $obj->Obtain('Little ox, I love you', '英语', '韩语');
echo '<pre>';
var_dump( $res );
````


### $this->Obtain()的参数说明：


``` 
$obj->Obtain(翻译内容, 起始语言, 目标语言);
```
